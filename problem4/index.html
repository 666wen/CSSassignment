A Beginner’s Guide to SVG Animation

Scalable Vector Graphics (SVGs) are more than just image files. As an XML application, SVGs contain markup that looks and works much like HTML. You can also use them in conjunction with CSS and JavaScript code. This makes it possible to animate SVG files, creating complex images that work well for web design and other dynamic environments.
